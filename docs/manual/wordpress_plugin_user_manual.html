CPD Search Users guide

Brief instructions on how to use the CPD search plugin.

The cpd search plugin enables 'commercial property agents' to provide a search feature on their wordpress website. Agents can choose to display a search of ONLY their own properties ('current instructions') or a full search of our entire database.

This user guide will show how to:

  • Search for commercial properties
  • View results
  • Add properties to a shortlist
  • Register interest in those properties
  • Verify a user
  • Download shortlist

Search form

The first screen a user will usually see is the 'search form'. Currently there are two options:

  • Selected properties
  • Full Search Form

Selected properties is a basic dropdown which will show a list of the 'property types' which the agent has properties available in. Simply select the type from the list and the results page will load.

Full search is a form with multiple inputs to filter results:

  • Sector - eg offices, industrial warehouses
  • Tenure - Leasehold, Freehold or both
  • Area - These are CPD defined areas
  • Location - Add your own location
  • Postcode
  • Size

Simply fill out the form or just click "search now" for a full search

Results view

This is a basic list of matching properties that displays some of the main details about each property.

Each property should have an 'add to shortlist' button which when clicked will add that property to the users shortlist.

Usually the shortlist should be visible in the results view and the user goes through and adds as many properties as they wish.

View / download Shortlist

Once the user is happy that they have added all the desired properties to their shortlist just click the 'download shortlist' link. To be able to see the full details of the properties the user will then be taken to a 'register user' form..

Register user contact details

This step is a basic form where the user should enter their contact details to be able to receive the downlooad of the shortlist and so the 'agent' can contact them about the properties they have added to the shortlist.

After the form is submitted the user will get a verification email.

View / Email shortlist

After contact details have been sent the 'shortlist' page will show. Here the user can choose to 'email list' or continue to search and add / remove more properties.

To receive the email the user will need to 'verify' their email address.

Verify user

User will be sent an email so that they can verify their email address. Just click the link in the email and the user will be taken to a verification success page. Once the email address is verified user will receive another email with the full property details.

Agent contact email

The website agent will also get an email notifying them that you the user has 'registered interest' in a property or properties.
